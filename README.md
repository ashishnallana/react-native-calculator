## Run Locally

Clone the project

```bash
  git clone https://github.com/ashishnallana/react-native-calculator.git
```

Go to the project directory

```bash
  cd react-native-calculator
```

Install dependencies 

```bash
  npm install
```

Run the app locally on your simulators

```bash
  npm start
```
